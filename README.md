# dtrader-crypto | CLI-бот для криптотрейдинга

Консольный (CLI) торговый бот для биржи Gate.io, написанный на Node.js и TypeScript. Приложение представляет собой полнофункциональный терминальный интерфейс (TUI) для мониторинга и управления торговыми стратегиями.

## 🚀 Особенности

- **Подключение в реальном времени:** Использует WebSocket API v4 биржи Gate.io для мгновенного получения обновлений цен и баланса.
- **Интерактивный интерфейс:** Построен с помощью библиотеки `terminal-kit` (v3), обеспечивая богатый пользовательский опыт прямо в консоли.
- **Виджетная система:**
  - **Хедер** с названием и виджетом статуса соединения/пинга.
  - **Основная область** для отображения балансов, цен и другой информации.
  - **Сайдбар** для вывода цветных, отформатированных логов (успех, ошибка, предупреждение, трейд).
  - **Футер** с полем для ввода команд.
- **Динамическое обновление:** Цены и балансы обновляются в реальном времени, с индикаторами роста (`▲`) или падения (`▼`) цены.
- **Модульная архитектура стратегий:** Легко добавлять новые торговые стратегии. Управление жизненным циклом стратегий (`start`, `pause`, `stop`) через команды в интерфейсе.
- **Безопасность:** Ключи API хранятся локально в файле `.env` и не попадают в систему контроля версий.
- **Строгая типизация:** Весь проект написан на TypeScript со строгим режимом, что обеспечивает надежность и предсказуемость кода.

## 🛠️ Установка и запуск

**1. Создайте файлы проекта:**
Создайте структуру папок и файлов, как указано выше, и скопируйте в них код из этого документа.

**2. Установите зависимости:**
Откройте терминал в корневой папке проекта (`dtrader-crypto`) и выполните команду:

```bash
npm install
```

3. Настройте ключи API:
   Скопируйте файл-пример .env.example в новый файл .env.
   code
   Bash
   cp .env.example .env
   Откройте файл .env в текстовом редакторе и вставьте ваши реальные API ключ и секрет от биржи Gate.io.
4. Запустите приложение:
   code
   Bash
   npm start
   Или напрямую через ts-node:
   code
   Bash
   npx ts-node src/index.ts
   ⌨️ Использование
   После запуска вы увидите интерфейс в терминале. В нижней части находится поле для ввода команд.
   Доступные команды:
   start <имя*стратегии>: Запускает указанную стратегию.
   pause <имя*стратегии>: Приостанавливает работу стратегии.
   stop <имя_стратегии>: Полностью останавливает и сбрасывает состояние стратегии.
   exit: Корректно завершает работу приложения.
   Пример:
   Для запуска демонстрационной стратегии введите команду и нажмите Enter:
   code
   Code
   start BTC_Cross_40k_70k
